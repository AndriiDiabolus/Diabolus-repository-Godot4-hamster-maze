## Описание игры
Браузерная игра "Хомяк в Лабиринте" (один HTML файл, без зависимостей).

### Технологии
- Чистый HTML5 + CSS + JavaScript (no framework)
- Canvas API для рендеринга
- Язык интерфейса: русский

### Константы карты
- `CELL = 36` — размер клетки в пикселях
- `COLS = 25`, `ROWS = 19` — размер лабиринта
- `HUD = 58` — высота HUD панели

### Механика игры
- **Цель:** собрать все орехи (🌰), появляются волнами каждые 25 секунд
- **Хомяк 🐹:** управляется стрелками / WASD
- **Синяя лама 🦙:** патрулирует лабиринт, слышит на 8 клеток, видит на 4; каждые 30 сек ускоряется
- **Красная лама 🦙:** появляется через 1 минуту с другого конца карты
- **Норы:** Пробел — спрятаться/вылезти, всего 3 норы
- **Кроличьи норы 🐰:** 2 пары порталов (золотые и голубые), после использования исчезают, через 1 мин появляются новые
- **Туман войны:** видимость ограничена
- **Управление:** WASD / стрелки, M — музыка, Enter — рестарт, Пробел — нора

### Алгоритмы
- **Генерация лабиринта:** recursive backtracking, мёртвые концы устраняются
- **Поиск пути врагов:** BFS

### UI / UX
- Тёмная тема: фон `#0d0d1a`, акцент `#ffd700` (золотой), синее свечение
- Адаптив для мобильных: d-pad кнопки (▲◄▼►), кнопка норы, кнопка музыки
- Splash-экран с правилами и таблицей рекордов
- Online таблица рекордов (leaderboard)
- Модалка ввода никнейма
- Модалка комментария к рекорду
- A2HS баннер (Add to Home Screen для мобильных)
